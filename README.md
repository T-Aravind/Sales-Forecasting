# 📊 Sales Forecasting Dashboard

A Machine Learning web application built using Streamlit that allows users to upload sales datasets and predict future sales.

## Features

- Upload CSV sales dataset
- Automatic data analysis
- Sales trend visualization
- Machine learning predictions
- Actual vs Predicted comparison

## Tech Stack

Python
Streamlit
Pandas
Scikit-learn
Matplotlib

## Installation

1. Clone repository

git clone https://github.com/T-Aravind/sales-forecast-dashboard

2. Install dependencies

pip install -r requirements.txt

3. Run application

streamlit run app.py
